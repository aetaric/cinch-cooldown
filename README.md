# Cinch::Cooldown

This library is used to add a global cooldown so that users are prevented from spamming the
channel.

## Installation

Add this line to your application's Gemfile:

    gem 'cinch-cooldown'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cinch-cooldown
